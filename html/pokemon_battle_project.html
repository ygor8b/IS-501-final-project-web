Pokémon and Move Classes is a Python project focused on implementing class design, encapsulation, and object interaction. It simulates a basic Pokémon battle system, where each Pokémon has stats, types, and can perform attacks using defined Move objects. The program uses class constructors, inheritance, and methods to demonstrate how objects interact and respond to game logic like damage calculation and health updates. This project reinforces core object-oriented programming skills in a fun, game-themed context.
