Movie Tracker is a command-line application that combines object-oriented programming with database management using the Peewee ORM library. Users can create and manage a watchlist of movies, mark them as watched, provide ratings, and filter for top-rated films. This project demonstrates skills in class design, input validation, CRUD operations, and database integration. The app features a clean user menu with options to add, view, update, and delete movie records stored in a local SQLite database.
